API com Mapa Interativo

Essa aplicação web foi também desenvolvida para fins acadêmicos, com foco em visualização geográfica e interatividade. O usuário pode explorar diferentes locais diretamente no mapa, selecionando opções através de uma lista lateral intuitiva. Cada ponto exibe informações relevantes em popups dinâmicos, e há integração com busca por CEP para localização precisa. O projeto utiliza Leaflet.js, OpenStreetMap e JavaScript assíncrono para garantir fluidez e responsividade na navegação!
